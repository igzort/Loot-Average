# Loot Average — Симулятор корейского рандома Lineage 2 

Олдскульный кликер-калькулятор, который воссоздает ту самую математику дропа и спойла из оригинальной Lineage 2 

## 🎮 О проекте
Программа рассчитывает, сколько ресурсов, кусков или рецептов вы получите за определенное количество убитых монстров. Проект создан для фанатов старой школы, которые хотят вспомнить цену хардкорного гринда или математически рассчитать свою удачу.

## 🚀 Основные возможности
* **Честный расчет**: Использует оригинальную формулу «корейского рандома» для вычисления выпадения предметов.
* **Прогноз времени**: Показывает, сколько реального времени уйдет на выбивание нужного шмота.
* **Интерактивный гринд**: Позволяет вручную «кликать» мобов или запускать макрос-автокликер.

* Pygame (или другая библиотека для UI)
